Wondercise web

推薦系統設計與 Onboarding 優化

提升用戶首次體驗與平台轉換率

Wondercise 是一款串流健身平台，致力於提供即時健身課程與個人化教練推薦。本次專案聚焦於提升新用戶的 onboarding 體驗，並設計一套推薦系統，協助使用者快速找到適合自己的健身課程，從而提高整體轉換率與黏著度。

🎯 專案目標

  • 精簡 onboarding 流程，提升註冊率
  • 設計推薦模組，引導使用者快速體驗
  • 提升首次課程播放與留存率

🧑 我的角色

  • 擔任 UX/UI 設計師與 Web 產品負責人
  • 主導使用者研究、資訊架構與設計產出
  • 與資料團隊共創推薦邏輯
  • 協作前後端將設計落地

❗ 問題發現

  • 有特定偏好教練的使用者活躍度與生命週期較高
  • 缺乏引導與內容推薦，使用動機不明確
  • 首次課程播放率 偏低，流失高

🔎 洞察重點

個人化推薦可有效激發「這就是為我設計的」感受，提升轉換率。

當用戶可以找到自己喜愛的教練時，有助於提高用戶訓練留存率。

🧠 專案反思

這次與資料科學團隊協作，讓我深刻理解推薦系統不只是演算法，而是 UX 如何塑造「這就是為我設計的」感受。

如果時間允許，我會進行長期追蹤，讓推薦模組在實際使用中持續進化，並找出不同族群的行為模式，實現更精準的推薦策略。
